
# The Drink Shoppe Logo Assets

This folder contains official logo assets for **The Drink Shoppe**.

## Overview
Sticker-style dachshund logo holding drinks with the text *The Drink Shoppe*.
Transparent background with white outline trim.

These assets are intended for:
- sticker printing
- web display
- repository branding
- potential merchandise

## Folder Structure

brand/logo/
  source/
  print/
  web/
  favicon/
  social/

### source
Original master assets.

### print
High-resolution PNG files intended for sticker printing.

### web
Optimized PNG versions for websites and GitHub.

### favicon
Small icons used for browser tabs and PWA installs.

### social
Preview images used for social sharing and link cards.

## Naming Convention

the-drink-shoppe-[asset]-[purpose]-[size].[ext]

Examples:
- the-drink-shoppe-logo_web_1024.png
- the-drink-shoppe-logo_print_300dpi_transparent.png
- the-drink-shoppe-icon_192.png

Rules:
- lowercase
- hyphen separated
- no spaces

## Print Notes

For sticker printing use files in the **print/** folder.

Do not:
- stretch or distort the logo
- remove the white trim outline
- upscale smaller web images for print
